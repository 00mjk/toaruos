ToAruOS Kernel and Userspace

ToAruOS is a hobby operating system project under development by a couple of UIUC computer science undergraduates.

Dependencies

nasm, gcc or clang, genext2fs to build RAM-disk, and an emulator which supports multiboot (or an equivalent set up with, ie, GRUB and/or physical hardware)

Install

`make && make run`, assuming dependencies are met.
